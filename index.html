Main Content

Welcome to WDD 130: Web Fundamentals! This course introduces you to the foundational concepts of web development, covering both front-end and back-end technologies. Throughout this semester, you will explore HTML, CSS, and JavaScript, learning how to create engaging and responsive websites.

By the end of this course, you will:

  • Understand the structure and semantics of HTML5.
  • Apply CSS for styling and layout purposes, including the use of Flexbox and Grid.
  • Implement JavaScript to enhance interactivity and functionality on web pages.
  • Gain hands-on experience with modern web development tools and practices.

This course is designed to provide you with practical skills that are essential for a career in web development. Whether you're aiming to become a front-end developer, back-end developer, or full-stack developer, the knowledge and experience gained here will serve as a solid foundation.

Throughout the semester, make sure to regularly check the navigation links above for weekly assignments, course resources, and updates. Feel free to reach out if you have any questions or need assistance. Let's embark on this journey of learning and building together!
